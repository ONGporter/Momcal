### momcal.vercel.app 서비스워커 이슈 — "지금 방식 유지"로 결정됨
momcal.vercel.app 접속자가 거의 없어서, 301 리다이렉트(momcal.vercel.app → momcal.app)는 그대로 유지하기로 결정. 참고로 momcal.vercel.app에 예전에 설치된 서비스워커는 리다이렉트 때문에 더 이상 자체 업데이트를 받을 수 없어 영구적으로 예전 버전에 머무름 — 코드로 고칠 수 없고 해당 사용자가 브라우저 사이트 데이터를 지워야만 해결됨. 감수하기로 함.

### 알아두면 좋은 것
- 정식 도메인은 `momcal.app` — `momcal.vercel.app`도 301 리다이렉트로 계속 접속은 가능(단, 서비스워커 이슈로 예전 방문자는 갱신 안 될 수 있음, 감수하기로 결정됨)
- 폰트 2종 체제: 제목/본문/버튼/로고는 "온글잎 박다현체", "정보 글"은 "오뮤 다예쁨체" — 둘 다 `font-weight: normal` 고정
- 하단 탭: 홈/캘린더/체크리스트/성장/설정 5개, "설정" 탭만 폭이 좁게 디자인됨
- 체크리스트 실시간 동기화(onSnapshot) 관련 코드를 만질 땐 `js/state.js`의 `subscribeToUserData` 주석 참고 — `hasPendingWrites`로 "내 저장 에코"와 "실제 원격 변경"을 구분해야 화면이 불필요하게 중복 렌더링되지 않음
- 캘린더 주간 뷰는 `S.calWeekRef`(YYYY-MM-DD) 자체 기준일을 갖고, `calMove()`가 뷰에 따라 분기(월간=월 단위, 주간=7일 단위)해서 이동시킴 (`js/calendar.js`)
- **v0.0.15**: 캘린더 주간 뷰의 시간대 그리드는 하루 칸 하나를 절대좌표 컨테이너로 만들어 시작~종료 시각에 맞는 높이의 블록 하나로 이어 그림(제목도 1번만 표시). 겹치는 일정끼리만 좌우로 트랙을 나눠 색을 분리함(`layoutDayTimedEvents`). 슬롯 경계(0/6/8/10/12/14/16/18/20/22시)는 실제 길이와 무관하게 화면에서 항상 같은 높이(`WEEK_ROW_H`)로 그려지고, 그 안에서는 비례 위치로 배치함 — 자세한 계산은 `js/calendar.js`의 `weekMinutesToY`/`layoutDayTimedEvents` 주석 참고
- **v0.0.15~16**: 주간 뷰에서 이유식 스티커는 헤더의 날짜 숫자 밑(`week-head-food-stickers`, "이유식" 글자 라벨 포함)에, 일반 스티커는 "종일" 행에 표시 — 월간 뷰(day-food-stickers)와 표시 위치 규칙은 같되 마크업만 다름
- 캘린더 셀에서 이유식 스티커(`FOOD_STICKER_SET`)는 다른 스티커의 "+N" 카운트와 완전히 분리되어 날짜 숫자 옆에 표시됨
- 캘린더 토·일·공휴일 날짜는 `--holiday-red` 변수로 붉은색 표시(`data/kr-holidays.js`로 공휴일 판별) — 공휴일 이름 자체는 화면에 안 띄움(아래 "다음 후보" 참고)
- **가족 그룹 공유(v0.0.12, v0.0.15에서 실사용 확인됨)**: `users/{uid}.familyId`가 있으면 `families/{familyId}` 문서를 데이터 소스로 씀. 자세한 구조는 `PROJECT_SPEC.md` "가족 그룹 공유" 섹션 참고. 가족 그룹 생성/참여/탈퇴는 전부 `location.reload()`로 마무리함(가족 전환 시 캐시된 파생 상태가 꼬이는 걸 막기 위한 의도적 선택)
- **v0.0.16**: "내 일정"(custom)으로 직접 추가하는 일정은 일정 추가 폼에서 직접 색상을 고를 수 있음(`ev.color` 필드, `getEvDisplayColor()`가 우선 적용) — 그 외 종류(필수/추천/접종/정부지원)는 기존처럼 캘린더 하단 범례에서 카테고리 공통 색을 바꿈. "내 일정"은 범례에서 제외됨
- **v0.0.16**: 캘린더 안 일정 글자(.ev-line/.week-ev-block 등, px 고정이라 앱 전체 글자 크기 설정의 영향을 안 받음)만 따로 조절하는 설정 추가(`js/calFontSize.js`, `data-cal-fontsize` 속성) — 설정 탭 + 캘린더 툴바 "⚙️" 바로가기에서 접근 가능
- **v0.0.16**: 홈 대시보드 카드 순서를 원래대로(오늘·다음 일정·체크리스트·성장·팁) 되돌리고 "최근 접종" 카드는 제거함. 체크리스트 카드는 "N/N 완료(필수)"와 "N/N 완료(전체)"를 한 줄에 나란히 표시(`js/ui.js`의 `dashChecklistCard`)하고, Perfect(🏅)/Master(👑)/Legend(🌈) 배지는 그 아래 작은 칩(`dash-badge-mini`)으로 표시함
